📊 Systeem Benchmark (p95)

Start een 5-minuten meting om het werkelijke verbruik van al je containers te bepalen. Dit kost tijdelijk CPU capaciteit, maar daarna blijven de resultaten statisch zichtbaar zonder belasting.

* Laatste run: onbekend. De dashboard tegels tonen de p95 waarden (95% van de tijd verbruikte de app minder dan dit).
* Note: CPU % wordt per core berekend (100% = 1 volledige core). Jouw systeem heeft een totale capaciteit van ?%.

💾 Backup & Restore

Exporteer al je instellingen (apps, thema, widgets, weather config) als JSON. Importeer een backup om te restoren of te migreren naar een ander apparaat.

🟦 Synology Apps▾

Voeg native Synology Package Center apps toe (die niet via Docker draaien). Selecteer welke je hebt en klik Toevoegen.

DSM poort heeft alleen effect op apps die DSM gebruiken (port 5000 in de lijst). Plex (32400), Jellyfin (8096), SABnzbd (8080) houden hun eigen poort.
